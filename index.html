Academic Projects

Library Management System (Object Oriented Programming)

In this project I implemented the management system of a university library in my Object Oriented Programming (OOP) course. I used Classes, SFML graphics to create a beautiful GUI in which all the facilities of a library were available e-g Books Available, No. Of Books, Student Book Issuing Record etc.

Traffic Light Control System (Digital Logic Design

In this project I implemented the four way traffic control system (Hardware Based) in my DLD course, I used different electronic components like Diodes,Resistors,Timers for controlling the glowing time of LEDs (used as Traffic Lights. )

Audio Modifier/Mixer (System Programming)

In my System Programming course, I made a program that was able to combine two audio files (of any format), I implemented this program using POLL and FILE I/O. In this project data from two audio files were read into a buffer and then were written to another new file.

Projectile Motion of a Cricket Ball(Signals and Systems)

For this project I used the reference of cricket to demonstrate the concept of Projectile Motion(using Matlab ). In this project the user had to input speed of ball, height of the bowler, pitching angle after that the program using certain formulas calculated the angle of ball, its height and speed after pitching, was able to show whether the ball hit the wickets or was a no ball or was wide ball

Pulse Code Modulation (Digital Signals Processing)

The purpose of this program was to convert an Analog Signal to Digital Signal For this project I used Matlab. In this project an input voice signal was given which was processed by the program using Sampling Method and then was converted to a Digital Signal as an output.
